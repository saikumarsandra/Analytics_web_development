JavaScript Functions

This page explains the different types of functions in JavaScript.

1. Function Declarations

Function declarations are the most common way to define a function. They are hoisted, which means they can be called before they are defined in the code.

        
            function greet(name) {
                return "Hello, " + name + "!"; 
            }

2. Function Expressions

Function expressions are defined as part of a larger expression, such as a variable assignment. They are not hoisted, so they cannot be called before they are defined.

        
            const greet = function(name) {
                return "Hello, " + name + "!"; 
            };

3. Arrow Functions

Arrow functions are a more concise way to write functions in JavaScript. They are especially useful for simple functions and when you want to preserve the value of `this`.

        
            const greet = (name) => {
                return "Hello, " + name + "!"; 
            };

4. Anonymous Functions

Anonymous functions are functions that do not have a name. They are often used as arguments to other functions or as immediately invoked function expressions (IIFEs).

        
            setTimeout(function() {
                console.log("This is an anonymous function!");
            }, 1000);

5. Immediately Invoked Function Expressions (IIFEs)

IIFEs are functions that are defined and immediately executed. They are often used to create a new scope and avoid polluting the global namespace.

  
        
            (function() {
                console.log("This is an IIFE!");
            })();

6. Generator Functions

Generator functions are a special type of function that can be paused and resumed. They are defined using the `function*` syntax and use the `yield` keyword to pause execution.

        
            function* generator() {
                yield "Hello";
                yield "World";
            }

7. Async Functions

Async functions are a special type of function that allows you to write asynchronous code in a more synchronous manner. They are defined using the `async` keyword and use the `await` keyword to wait for promises to resolve.

        
            async function fetchData() {
                const response = await fetch('https://api.example.com/data');
                const data = await response.json();
                return data;
            }

Conclusion

JavaScript offers a variety of ways to define functions, each with its own use cases and advantages. Understanding these different types of functions can help you write more efficient and readable code.
